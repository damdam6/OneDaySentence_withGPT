The diagram illustrates the following components and their interactions:
- **MainPage**: Represents the main page developed using React.
- **EmojiSelectionButton**: A button that users can click to select an emoji.
- **EmojiChoiceView**: A view that displays the available emoji options for users to choose from.
- **NewPage**: Represents the new page that appears after an emoji is selected, developed using React.
- **DaySentenceView**: A view that displays the generated sentence for the day.
- **BackToMainButton**: A button that allows users to navigate back to the main page.
- **UserProfile**: Allows users to save their favorite sentences and view their emoji history.
- **FeedbackSystem**: Enables users to rate or provide feedback on the generated sentences and shuffle for a new sentence.
- **NotificationSystem**: Sends daily prompts to users.
- **SocialSharing**: Allows users to share their sentence on social media platforms.
- **Analytics**: Tracks user interactions and popular emojis.
